# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in AgentGuard, please report it responsibly:

1. **Do NOT open a public GitHub issue.**
2. Use [GitHub Private Vulnerability Reporting](https://github.com/dockfixlabs/agentguard/security/advisories/new)
3. Or email: security@dockfixlabs.dev
4. Include a clear description and steps to reproduce
5. We will respond within 72 hours

## Disclosure

We follow responsible disclosure. Once a fix is released, we will publish a security advisory with full credits.
